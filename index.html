Lógica de Programação

Lógica de programação é definir os passos e a ordem necessária para alcançar um objetivo

Escopo global: É quando uma variável é declarada FORA DE UMA FUNÇÃO. Isso significa que qualquer variável declarada fora de um bloco (função) está disponível para utilizada em todo o código.

Escopo local: É quando uma variável é declarada DENTRO DE UMA FUNÇÃO (bloco)

Variável pode ser entendida como uma caixinha/gaveta, onde os dados são armazenados temporariamente ou definitivamente. (Guardamos as informações, ex: nome, idade, frase...)
São usadas como nomes simbólicos para os valores da sua aplicação. Os identificadores, nome das variáveis, obedecem determinadas regras.

  • VAR: Antes da criação do LET, havia problemas associados a declaração com var. Por isso, foi necessário criar a variável LET. VAR, escopo global.
  • LET: Declara uma variável local, é a mais utilizada. Elas podem ser atualizadas, mas não podem ser declaradas novamente.
  • CONST: Não pode ser atualizada e nem declarada novamente, porque tem valores constantes.
